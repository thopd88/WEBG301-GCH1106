# New Laravel Project
## Requirements
- PHP >8.0
- Composer
- MySQL
- Visual Studio Code
- Git
## Create Project
```composer create-project laravel/laravel Book```
## Run Project
```php artisan serve```
## Create Model
```php artisan make:model Book -m```
## Update Database
```php artisan migrate```
## Refresh Database
```php artisan migrate:refresh```
