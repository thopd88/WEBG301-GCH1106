# New Laravel Project
## Requirements
- PHP >8.0
- Composer
- MySQL
- Visual Studio Code
- Git
## Create Project
```composer create-project laravel/laravel Book```
## Run Project
```php artisan serve```
## Create Model
```php artisan make:model Book -m```
## Update Database
```php artisan migrate```
## Refresh Database
```php artisan migrate:refresh```

# Clone Project
## Clone Project
```git clone```
## Install Dependencies
```composer update```
## Create .env file and modify database settings
```cp .env.example .env```
## Generate Key
```php artisan key:generate```
## Migration
```php artisan migrate```
## Run Project
```php artisan serve```
